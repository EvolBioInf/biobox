  \section*{Introduction}
  Given a set of numbers, the program \texttt{var} computes their mean,
  variance, and standard deviation. The variance is calculated using the
  numerically stable two-pass method. The numbers are read from a single
  column, one number per line.

#+begin_src latex
  The variance and standard deviation are computed in the second pass.
#+end_src
#+begin_src go <<Second pass, Ch.~\ref{ch:var}>>=
  var variance float64
  for _, v := range data {
	  d := ave - v
	  variance += d * d
  }
  variance /= float64(n - 1)
  sdev := math.Sqrt(variance)
#+end_src
