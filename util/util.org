#+begin_src latex
  !\texttt{PrintInfo} prints a program's name, version, and compilation
  !date. It also prints the author, email address, and license of the
  !\texttt{biobox} package. Then it exits.
  To achieve this, we wrap the generic
  function for printing program information from package \texttt{clio}.
#+end_src
#+begin_src go <<Functions, Ch.~\ref{ch:uti}>>=
  func PrintInfo(name string) {
	  clio.PrintInfo(name, version, date, author, email, license)
	  os.Exit(0)
  }
#+end_src
#+begin_src latex
  We import \texttt{clio} and \texttt{os}.

#+end_src
